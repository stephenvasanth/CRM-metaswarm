## Redis Cache-First Pattern (NFR-P03 — hard constraint)

ALL service reads must:
1. Check Redis (`redisTemplate.opsForValue().get(cacheKey)`) → if non-null, return cached value
2. On cache miss: fetch from DB → `redisTemplate.opsForValue().set(key, value, 24, TimeUnit.HOURS)` → return
3. ALL writes (POST/PUT/DELETE/PATCH) call `invalidateXxxCache()` which does `keys = redisTemplate.keys("xxx:*"); if (keys != null && !keys.isEmpty()) redisTemplate.delete(keys)`

Both cache-hit and cache-miss branches MUST be tested. The `@BeforeEach` stub `when(redisTemplate.opsForValue()).thenReturn(valueOps)` plus `@MockitoSettings(strictness = Strictness.LENIENT)` is the working pattern.
